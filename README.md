# Brain Freeze Arena - オンライン対戦

氷上の最短一手を見つけろ！スマホ対応のオンライン対戦パズルゲーム

## セットアップ

### 1. 依存関係のインストール

```bash
cd server
npm install
```

### 2. サーバーの起動

```bash
npm start
```

または開発モード（自動再起動）:

```bash
npm run dev
```

サーバーは `http://localhost:3000` で起動します。

### 3. ゲームにアクセス

ブラウザで以下のURLを開きます：

- **待合室**: http://localhost:3000/lobby.html
- **ソロ練習**: http://localhost:3000/index.html

## スマホからアクセスする方法

### 1. 同じWi-Fiネットワークに接続

PCとスマホを同じWi-Fiに接続します。

### 2. PCのIPアドレスを確認

**macOS:**
```bash
ifconfig | grep "inet " | grep -v 127.0.0.1
```

**Windows:**
```bash
ipconfig
```

例: `192.168.1.100`

### 3. スマホのブラウザでアクセス

```
http://192.168.1.100:3000/lobby.html
```

### 4. Socket.IOの接続先を変更

`js/lobby-client.js` と `js/online.js` の以下の行を修正：

```javascript
// 変更前
const socket = io('http://localhost:3000');

// 変更後（PCのIPアドレスに置き換え）
const socket = io('http://192.168.1.100:3000');
```

## 遊び方

### オンライン対戦

1. **待合室にアクセス**: `lobby.html` を開く
2. **プレイヤー名を入力**: 好きな名前を入力してロビーに入る
3. **部屋を作成 or 参加**:
   - 「部屋を作成」ボタンでホストになる
   - 既存の部屋に「参加」ボタンで参加
4. **準備完了**: ゲストは「準備完了」ボタンを押す
5. **ゲーム開始**: ホストが「ゲーム開始」ボタンを押す

### ソロ練習

1. `index.html` を開く
2. 「ソロで練習したい場合はこちら」をクリック
3. 盤面が表示されたら、ロボットをクリックして選択
4. 矢印キーでロボットを移動
5. 同じ色のゴール（★）に到達させる

## ゲームルール

- ロボットは**壁または他のロボットにぶつかるまで滑り続ける**
- 1回の直進を1手とカウント
- 指定された色のロボットを同じ色のゴールに到達させる

### Quick Mode（5本先取）
- 正解すると1本獲得
- 先に5本獲得したプレイヤーが勝利

### Score Mode（10ラウンド）
- 10ラウンド実施
- 得点 = 残り時間(秒) × 宣言手数
- 総得点が最も高いプレイヤーが勝利

## トラブルシューティング

### サーバーが起動しない

- Node.jsがインストールされているか確認: `node -v`
- 依存関係を再インストール: `npm install`

### スマホから接続できない

- PCとスマホが同じWi-Fiに接続されているか確認
- ファイアウォールでポート3000が開いているか確認
- Socket.IOの接続先URLが正しいか確認

### ゲームが同期されない

- ブラウザのコンソールでエラーを確認
- サーバーのログを確認
- ページをリロードして再接続
