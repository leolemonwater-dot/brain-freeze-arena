# Brain Freeze Arena - システム要件定義書

最終更新: 2026-05-31
ステータス: ✅ 確定（設計フェーズへ進んでよい）

---

## 1. 対象環境

| 項目 | 仕様 |
|------|------|
| デバイス | スマホ（縦画面）メイン |
| スクロール | なし（1画面に収める） |
| 対応ブラウザ | Chrome / Safari |
| 効果音・BGM | 今後追加予定（設計時に拡張しやすい構造にしておく） |

---

## 2. 共通UI仕様

### 戻るボタン
- **全画面の左上**に ◀ ボタンを配置する
- タイトル画面のみ戻るボタンなし（起点のため）
- 各画面の戻り先は「画面一覧」に記載

### 確認ダイアログ
- ゲーム中に ◀ ボタンを押した場合、確認ダイアログを表示する
  - オフライン：「ゲームを終了しますか？」YES → 画面④ / NO → 戻らない
  - オンライン：「リタイアしますか？」YES → 切断扱い → 画面③b / NO → 戻らない
  - ソロ練習：「ゲームを終了しますか？」YES → 画面① / NO → 戻らない

---

## 3. 画面一覧・遷移

### 画面①：タイトル画面
**URL:** `index.html`

**表示要素:**
- ロゴ画像（ペンギン）
- オンライン対戦ボタン
- オフライン対戦ボタン
- ソロ練習ボタン

**戻るボタン:** なし

**遷移先:**
- オンライン対戦ボタン → 画面②
- オフライン対戦ボタン → 画面④
- ソロ練習ボタン → 画面⑥（宣言パネルなし）

---

### 画面②：名前入力画面（オンライン）
**URL:** `lobby.html`

**表示要素:**
- ロゴ画像
- プレイヤー名入力フォーム
- 「ロビーに入る」ボタン

**戻るボタン:** ◀ → 画面①

**遷移先:**
- ロビーに入る → 画面③

---

### 画面③：待合室（オンライン）
**URL:** `lobby.html`（同一ページ内で切り替え）

**表示要素:**
- ロゴ画像
- プレイヤー名表示（「ようこそ、〇〇さん」）
- 「更新」ボタン（押したタイミングで部屋一覧を再取得）
- 部屋一覧リスト
  - 部屋名（「〇〇の部屋」）
  - モード表示（Quick / Score）
  - 参加人数（例: 2/4）
  - 状態バッジ（待機中 / 満員 / ゲーム中）
  - 「参加」ボタン（待機中かつ満員でない場合のみ有効）
- 「部屋を作成」ボタン

**戻るボタン:** ◀ → 画面①

**遷移先:**
- 参加ボタン → 画面③b
- 部屋を作成 → 画面③b

**備考:**
- 部屋一覧はリアルタイム自動更新ではなく、「更新」ボタンで手動更新
- 満員・ゲーム中の部屋も一覧に表示する（参加ボタンは無効化）

---

### 画面③b：部屋内待機（オンライン）
**URL:** `lobby.html`（同一ページ内で切り替え）

**表示要素:**
- 部屋ID
- ゲームモード表示（ホストが設定、参加者は閲覧のみ）
- 参加プレイヤー一覧（名前・ホスト / 準備完了 / 待機中バッジ）
- ホストのみ：「ゲーム開始」ボタン（全員準備完了で有効化）
- ゲストのみ：「準備完了」「準備キャンセル」ボタン
- 「退出」ボタン
- 「メニューへ戻る」ボタン → 画面①（タイトル）

**戻るボタン:** ◀ → 画面③（退出と同じ動作）

**遷移先:**
- ゲーム開始（ホスト） → 画面⑥
- 退出 / ◀ → 画面③
- メニューへ戻る → 画面①

**ホスト交代:**
- 待機中・ゲーム中を問わず、ホストが退出・切断した場合、入室が最も早いプレイヤーが新ホストになる

---

### 画面④：オフライン設定画面
**URL:** `index.html`（同一ページ内で切り替え）

**表示要素:**
- ゲームモード選択（Quick Mode / Score Mode）
- プレイヤー名入力・追加（2〜4人）
- プレイヤー一覧（削除ボタン付き）
- 「ゲーム開始」ボタン
- ※ゲーム終了後に戻ってきた場合、前回のプレイヤー名・人数を保持した状態で表示

**戻るボタン:** ◀ → 画面①

**遷移先:**
- ゲーム開始 → 画面⑥

---

### 画面⑤：結果画面
**URL:** `index.html` / `lobby.html`（同一ページ内で切り替え）

**表示要素:**
- 優勝プレイヤー名（大きく表示）
- 同点の場合は「同率1位」として複数名を表示
- 全プレイヤーの最終スコア一覧（順位付き）
  - Quick Mode：獲得本数
  - Score Mode：獲得得点
- ボタン
  - オフライン：「もう一度」→ 画面④（名前・人数保持）
  - オンライン：「ルームへ戻る」→ 画面③b（スコアリセット）
- 各自のタイミングでボタンを押して遷移（全員が揃うまで待たない）

**戻るボタン:** ◀ → 画面①（タイトル）

---

### 画面⑥：ゲーム画面
**URL:** `index.html`（同一ページ内で切り替え）

**表示要素（ヘッダー）:**
- ◀ ボタン（左上）
- ラウンド情報（例: Quick Mode / Score Mode 3/10ラウンド）
- タイマー（思考中残り秒数 / アディショナル残り秒数 / 解答フェーズ残り秒数）
- スコアボード（プレイヤーカード一覧）
  - オフライン：カードをクリックして操作プレイヤーを選択できる
  - オンライン：カードのクリック選択は不要（各自のデバイスで操作）
  - オンライン：宣言済みの場合、カードに宣言手数を表示する
- ステータステキスト
  - オンライン解答フェーズ中：解答者の画面に「あなたの番です」を表示

**表示要素（盤面）:**
- 12×12グリッド
- 壁（L字・I字）
- 中央2×2ブロック
- ロボット4色
- ゴール（★）

**表示要素（下部固定パネル）:**
- 「手数を宣言する」パネル
  - 手数入力
  - 「宣言」ボタン（宣言済みの場合グレーアウト）
  - 「パス」ボタン
- 解答フェーズ中：手数カウンター（現在手数 / 宣言手数）

**遷移先:**
- ゲーム終了 → 画面⑤（結果画面）
- ◀ ボタン → 確認ダイアログ（上記「共通UI仕様」参照）

## 4. ゲームルール仕様

### 思考フェーズ
- タイムリミット：60秒
- 60秒以内に誰も宣言しなかった場合 → ラウンドスキップ（次の盤面へ）

### アディショナルタイム
- 最初の宣言から30秒
- 切断プレイヤーは「0秒でパス宣言」と同等の扱い

### 解答フェーズ（オンライン限定）
- 解答者に **30秒のタイムリミット** を設ける
- 30秒超過 → 不正解扱い、強制的に次の解答者へ移行
- タイマーはヘッダーに表示

### 解答フェーズ（オフライン）
- タイムリミットなし

### ゲーム終了条件
| モード | 終了条件 |
|--------|---------|
| Quick Mode | 先に5本獲得 または 残り1人になった時点で即勝利 |
| Score Mode | 10ラウンド終了後に最高得点 または 残り1人になった時点で即勝利 |
| ソロ練習 | 終了条件なし（◀ボタンで終了） |

### 同点の扱い
- 結果画面で「同率1位」として複数名を表示

---

## 5. ゲーム終了後の処理

| モード | 処理 |
|--------|------|
| オフライン対戦 | 画面⑤ →「もう一度」で画面④。プレイヤー名・人数を保持 |
| オンライン対戦 | 画面⑤ →「ルームへ戻る」で画面③b。スコアリセット。全員が再度準備完了 → ホストがゲーム開始 |
| ソロ練習 | 盤面を再生成してそのまま続ける（結果画面なし） |

---

## 6. オンライン対戦の仕様

| 項目 | 仕様 |
|------|------|
| 最大人数 | 4人 |
| 最小人数 | 2人 |
| 部屋の作成 | ホストが作成、ランダムID発行 |
| モード選択 | ホストのみが決定。参加者は閲覧のみ |
| 部屋への参加 | 待合室の一覧から選択するのみ |
| 部屋一覧の更新 | 手動（「更新」ボタンを押したタイミングで再取得） |
| 部屋の表示 | 満員・ゲーム中の部屋も表示（参加ボタンは無効化） |
| ゲーム開始条件 | 全員準備完了 + ホストが開始ボタン |
| 盤面の同期 | ホストが生成して全員に配信 |
| ロボット操作 | 解答フェーズ中、解答者本人のデバイスのみ操作可能 |
| ロボット同期 | 解答者の操作はリアルタイムで全員の画面に反映 |
| 解答フェーズ表示 | 解答者の画面に「あなたの番です」を表示 |
| 解答タイムリミット | 30秒。超過したら不正解扱いで次の解答者へ強制移行（オンラインのみ） |
| 宣言の同期 | リアルタイムで全員に反映。プレイヤーカードに宣言手数を表示 |
| 宣言済みボタン | 宣言済みの場合、宣言ボタンをグレーアウト |
| 切断の判定 | 切断から20秒以内に再接続しなければ切断とみなす |
| 切断時の扱い | アディショナルタイム中は「0秒でパス宣言」と同等 |
| 切断時の処理 | 残ったプレイヤーでゲーム続行。残り1人になったら即勝利 |
| 復帰時の扱い | 20秒以内に再接続した場合はゲームに復帰。ペナルティ状態は切断前の状態を引き継ぐ |
| リタイア | ◀ボタン →「リタイアしますか？」YES で切断扱い |
| ホスト交代 | 待機中・ゲーム中を問わず、ホストが退出・切断したら入室が最も早いプレイヤーが新ホストになる |
| ゲーム終了後 | 画面⑤（結果画面）→「ルームへ戻る」で画面③b（スコアリセット） |
| 再戦 | 全員が準備完了 → ホストがゲーム開始ボタンで再戦 |

## 7. 画面遷移まとめ

```
画面①（タイトル）
  ├─ オンライン対戦 ──→ 画面②（名前入力）◀→①
  │                         └─→ 画面③（待合室）◀→①
  │                                   └─→ 画面③b（部屋内待機）◀→③
  │                                       │  └─ メニューへ戻る →①
  │                                       └─→ 画面⑥（ゲーム）◀→③b※確認ダイアログ
  │                                                   └─ 終了後 →画面⑤（結果）◀→①
  │                                                                 └─ ルームへ戻る →③b
  │
  ├─ オフライン対戦 ─→ 画面④（設定）◀→①
  │                         └─→ 画面⑥（ゲーム）◀→④※確認ダイアログ
  │                                     └─ 終了後 →画面⑤（結果）◀→①
  │                                                     └─ もう一度 →④（名前・人数保持）
  │
  └─ ソロ練習 ────→ 画面⑥（ゲーム・宣言パネルなし）◀→①※確認ダイアログ
                              └─ 終了後 → 盤面再生成してそのまま
```
